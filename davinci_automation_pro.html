🤖 [혁신] 다빈치 리졸브 타임라인 원클릭 파이썬 스크립트 사용법

더 이상 마우스로 60개 이미지와 LTX 비디오, 오디오, SRT를 일일이 드래그하여 타임라인에 올리지 마세요!

  1. 폴더 정렬: 작업 폴더 하나에 오디오(audio.wav 또는 audio.mp3), 자막(subtitles.srt), 그리고 씬 이미지/영상들(scene_01.png, scene_03.mp4 등)을 모아둡니다.
  2. 이 웹 도구에서 SRT 파일을 올린 후 [🤖 다빈치 타임라인 자동배치 스크립트 (.py) 다운로드]를 클릭합니다.
  3. 다빈치 실행: 다빈치 리졸브를 열고 프로젝트와 빈 타임라인을 하나 만듭니다.
  4. Console 창 열기: 다빈치 상단 메뉴 [Workspace] ➔ [Console]을 누르거나, 우측 콘솔 패널을 엽니다.
  5. 상단 탭에서 [Py3] (Python 3)을 선택합니다.
  6. 다운받은 davinci_auto_batch.py 파일을 텍스트 에디터로 열어 내용 전체 복사 후, 다빈치 Console 입력창에 붙여넣기(Ctrl+V) ➔ Enter를 누릅니다.
  7. 결과: 0.1초 만에 오디오, 자막, PNG 이미지, LTX MP4 비디오가 자막 타임코드 길이에 맞춰 칼같이 배치 및 정렬됩니다!

📂 씬 번호 일관성 규칙 (PNG와 LTX MP4 혼합 배치)

자동 스크립트가 파일을 정확히 인식하도록 폴더 안의 에셋 파일명을 아래 규격대로 유지해주세요.

E:/MyProjectFolder/ ├── audio.wav (또는 audio.mp3 - 음성 파일) ├── subtitles.srt (자막 파일) ├── scene_01.png (1번 씬 이미지) ├── scene_02.png (2번 씬 이미지) ├── scene_03.mp4 (3번 씬 LTX 생성 비디오 - 자동으로 MP4 우선 선택!) ├── scene_04.png (4번 씬 이미지) └── scene_05.mp4 (5번 씬 LTX 생성 비디오)

💡 팁: 동일 씬 번호에 .mp4와 .png가 동시에 존재하는 경우, 스크립트가 MP4 비디오를 우선 적용합니다!

🛠️ 다빈치 리졸브 프레임 레이트(FPS) 동기화

편집 전 다빈치 리졸브의 프레임과 이 도구의 FPS를 똑같이 맞춰야 1프레임의 오차도 없이 타임라인이 일치합니다.

  1. 다빈치 우측 하단 톱니바퀴 아이콘 (Project Settings) 클릭 (Shift + 9)
  2. [Master Settings] ➔ [Timeline frame rate] 확인 (유튜브 추천: 24 fps 또는 30 fps)
  3. 이 웹 도구의 하단 설정 패널에서도 동일한 FPS를 선택해 주세요.

📌 오디오 트랙 마커 가이드 심기

파이썬 스크립트 외에 수동 보조 작업용으로 마커 가이드선이 필요할 때 사용합니다.

  1. 분석 후 [🎬 정밀 마커(CSV) 다운로드]를 누릅니다.
  2. 다빈치 미디어 풀(Media Pool) 속 오디오 녹음 파일(MP3/WAV)을 우클릭합니다.
  3. [Import Markers from CSV...]를 선택하고 CSV 파일을 지정합니다.
  4. 오디오 클립에 알록달록한 마커가 심겨 클립 이동 시 가이드선이 따라 움직입니다.

🎥 Dynamic Zoom & 인스펙터 키프레임 연출법

💡 방법 A. 클릭 한 번으로 끝내기: Dynamic Zoom
  1. 타임라인 비디오 클립 클릭 ➔ 우측 상단 Inspector 창 열기
  2. [Dynamic Zoom] 오른쪽 스위치 ON (기본: 서서히 축소)
  3. 반대로 서서히 확대하려면 하단 Swap 버튼 클릭
🛠️ 방법 B. 정밀 키프레임 (Zoom & Position)

클립 첫 프레임에서 Inspector의 Zoom / Position 옆 ◆ 다이아몬드 클릭 ➔ 마지막 프레임으로 이동하여 수치 조절 (예: 1.0 ➔ 1.25).

🎙️ 저음량 마이크(-20dB) 심폐소생술 & 오디오 더킹

  1. Fairlight 탭 이동: 우측 상단 Effects ➔ Noise Reduction 드래그 ➔ Auto Speech Mode 클릭 (전기 노이즈 즉시 차단)
  2. Dynamics Gate: Threshold를 -35 dB ~ -45 dB로 설정하여 무음 구간 완전 정적 처리
  3. Normalize Audio: 오디오 우클릭 ➔ [Normalize Audio Levels...] ➔ True Peak -2.0 dB 설정

⚡ 에디터 필수 단축키

기능 단축키 설명

정밀 자르기 Ctrl + B 현재 플레이헤드 위치를 즉시 자릅니다.

🚀 60씬 양산 공장형 템플릿 연출법

1~8번 인트로 씬은 LTX MP4 비디오를 그대로 활용하고, 9번 씬부터 파워 빈(Power Bins) 조정 클립 4가지(Zoom In ➔ Zoom Out ➔ Pan L ➔ Pan R)를 바둑판처럼 번갈아 배치하세요. 1분 만에 60씬 역동적 연출이 완성됩니다.

🔥 LTX 모션 영상화 클라이맥스 스마트 추천 패널

영상의 초반 1~8번 인트로뿐만 아니라, **대본 중간중간 시청자 이탈을 방지할 극적 클라이맥스 씬(싸움, 무기, 호랑이, 벼락, 비명, 절규 등)**을 AI가 자동 스캔하여 LTX 비디오화 추천 씬으로 선정해 드립니다.

  1. 자막 분석 후 결과창 상단에 나타나는 [🔥 AI 모션 추천 씬 일괄 반영] 버튼을 누릅니다.
  2. 설정창의 [🎥 LTX 비디오 적용 씬 번호]에 인트로(1-8) 및 본문 추천 씬 번호가 자동으로 합산 입력됩니다.
  3. 파이썬 배치 스크립트 실행 시, 추천 씬들에 생성된 .mp4 동영상 파일이 다빈치 타임라인 상에서 자동으로 이미지 대신 영상 트랙으로 정밀 정렬됩니다!
