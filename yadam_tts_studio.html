야담 TTS 스튜디오 완벽 사용 설명 & 음질 튜닝 가이드

1. AI 음성 엔진 선택 가이드 (Google Cloud vs ElevenLabs vs VREW)

  • Google Cloud TTS (기본 권장): 매월 약 100만~400만 자까지 무료 이용이 가능하며, 안정적인 속도로 대용량 야담 원고를 합성합니다.
    - Neural2: 차세대 딥러닝 고성능 음성. 깔끔하고 딕션이 명확한 표준 오디오북 나레이터입니다.
    - WaveNet: 딥마인드 파동 합성 기술. 무게감 있는 사극/역사 다큐 나레이션에 매우 적합합니다.
  • ElevenLabs AI (초현실주의 감정 성우): 인간 성우 수준의 미세한 감정, 호흡, 억양 표현이 지원됩니다. (Profile API Key 입력 시 사용 가능)
  • VREW 대비 차이점 및 장점: VREW는 월 사용량이 금방 소진되는 구독 형태지만, 본 스튜디오의 Google Cloud API 연동 방식은 비용 효율이 월등하여 장편 오디오북과 유튜브 대형 야담 채널 운영 시 비용 부담이 거의 없습니다.

2. 마스터링 이퀄라이저 & 음질 깨짐(지직거림) 방지

  • ✨ 깔끔한 원음 (기본값 추천 ★★★): 과도한 저음 부스트 없이 가장 맑고 깨끗한 원본 구글/ElevenLabs 오디오를 출력합니다. 음성 찢어짐이나 깨짐이 전혀 없습니다.
  • 🎧 고음질 오디오북 / 📺 뉴스 해설: 이어폰 감상 및 대사 전달력을 높이기 위한 명확한 선명 톤입니다.
  • 📻 팟캐스트 중저음: 묵직한 중저음 울림을 주지만, 재생 기기/이어폰에 따라 울림이 과하면 '깔끔한 원음'으로 변경하시면 원시 맑은 음성으로 복구됩니다.
  • 음계 피치 튜닝 (-3.5 ~ +2.0 semitone): 10대 야담 성우 프리셋의 피치 조절 범위를 음색 깨짐 없이 자연스럽고 선명한 극적 영역으로 최적 보정했습니다.

3. 미리듣기(Preview)와 실제 생성(Generate)의 차이점

  • 미리듣기: API 키 미등록 시 브라우저 내장 보정 모드로 재생되며, 등록 시 1문장 즉시 합성으로 샘플을 테스트합니다.
  • 생성하기(Generate): 클라우드 AI 고음질 MP3 파일 생성과 동시에 센턴스 타임코드 자막(SRT/VTT)이 한 번에 자동 추출됩니다.

4. 자막 타임라인 내보내기 & SSML 태그 극적 연출

  • 영상 편집 프로그램 연동: SRT, VTT, 다빈치 리졸브/프리미어 전용 자막 버튼을 통해 영상 편집 프로그램 트랙에 바로 드래그하여 배치할 수 있습니다.
  • SSML 태그 사용: 원고 내에 <pause time="0.8s">, <sigh>, <laugh> 등의 태그를 클릭해 넣으면 극적 연출감이 한층 상승합니다.
